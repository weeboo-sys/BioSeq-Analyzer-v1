# BioSeq Analyzer v1

A beginner bioinformatics project built using Python and Biopython for analyzing biological sequences from FASTA files.

---

## Features

- FASTA file parsing
- Sequence length calculation
- GC content analysis
- Nucleotide counting
- DNA → RNA transcription
- Reverse complement generation
- Protein translation

---

## Technologies Used

- Python
- Biopython
- Git & GitHub
- VS Code

---

## Sample Workflow

Input:
FASTA sequence file

Output:
- Sequence ID
- Sequence length
- GC content
- RNA sequence
- Reverse complement
- Protein sequence
